# Projeto Cadastro Simples

Este é um projeto inicial em Python desenvolvido para praticar a captura e exibição de dados no terminal.

## 🚀 Como funciona?

O script `index.py` executa uma tarefa simples:
1. Solicita que o usuário digite o seu **Nome**.
2. Solicita que o usuário digite o seu **E-mail**.
3. Exibe uma mensagem de confirmação no terminal com os dados recebidos.

### Exemplo de Uso:
Digite seu nome: Leonardo
Digite seu e-mail: leonardo@email.com

--- Cadastro Realizado ---
Nome: Leonardo
E-mail: leonardo@email.com
